Developer Summary: E-PlantShopping
Author: Brian McCarthy
Date: May 2026

Technologies Used:
- React: The core library for building the user interface (JSX).
- Redux Toolkit: Used for state management, specifically for the shopping cart core logic (adding, removing, and updating quantities).
- React-Redux: For connecting React components to the Redux store.
- Tailwind CSS: For styling and responsive design.
- Lucide React: For consistent and beautiful iconography.
- Vite: The build tool and development server.

Programming Concepts:
- Component-Based Architecture: The app is divided into reusable UI components like AboutUs, CartItem, ProductList, etc.
- Functional Components & Hooks: Utilizing useState and useEffect for local state and side effects.
- State Management: Centralized store using Redux to ensure cart consistency across different views.
- Conditional Rendering: Dynamically switching between the Landing, Product Listing, and Cart views.
- Array Methods: extensively using map, filter, and reduce to manage and display plant data and calculate totals.

File Breakdown:
- App.jsx: The main entry point that manages the primary navigation state between the landing page and the shop.
- main.jsx: Boots the application and wraps it in the Redux Provider.
- App.css: Contains custom styles including the landing page background image.
- store.js: Configures the Global Redux store.
- CartSlice.jsx: Contains the 'slice' of state for the shopping cart, including reducers for adding/removing items and updating quantities.
- components/LandingPage.jsx: Welcomes users with an immersive "Welcome to E-PlantShopping" hero section.
- components/ProductList.jsx: Displays the catalog of plants grouped into categories.
- components/CartItem.jsx: The detailed shopping cart view with cost calculations, quantity controls, and a "Coming Soon" checkout alert.
- components/AboutUs.jsx: Shared component wrapped in an 'about-us-container' explaining the nursery's mission.

Application Functionality:
The E-PlantShopping app is a holistic e-commerce experience for plant lovers. It features a lush landing page branded as E-PlantShopping, a categorized product listing with a real-time updating cart icon, and a detailed checkout simulation. Users can browse unique categories (Fruit Plants, Flower Plants, Fruit Trees, Orchids, etc.), manage their select plants in a refined cart interface, and see instant total updates.
